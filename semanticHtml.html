Attributes

  • rel: in anchor element, for send information from go
  • rol: in div element, to assign a role according to the content it will have
  • download: in anchor element, When it has an image inside and the href is equal to the img element, then clicking on it can download it. It only works with the server's own resources
    example: Click on the image and it will download
  • type: in ol element, that is listed with numbers, letters, etc.
  • reversed: in ol element, to go in reversed
  • start: in ol element, to tell you where to start counting
  • pattern: in input type='tel' element, to add simple validation
  • loading: in img element, to load an image when needed or after everything has loaded
